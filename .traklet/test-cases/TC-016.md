{traklet:section:objective}
## Objective
A user can apply the FREE100 coupon code at checkout, see the total change to $0.00, complete checkout without payment information, and receive an active subscription.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in or completing checkout flow
- FREE100 coupon exists in database with 100% discount
- Coupon is active and not expired
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to `/checkout?plan=standard`
2. Verify checkout page shows Standard plan at $19.00
3. Locate coupon code input field (`data-testid="input-coupon"`)
4. Enter "FREE100"
5. Click "Apply Coupon" button (`data-testid="btn-apply-coupon"`)
6. Verify success message: "Applied FREE100"
7. Verify total updates to $0.00
8. Verify payment section hidden or not required
9. Click "Complete Checkout" button
10. Verify redirect to dashboard within 15 seconds
11. Verify subscription status shows "Standard - Active"
12. Verify no charge made (check database subscriptions table)
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- Coupon applies successfully
- Total calculates correctly: $19.00 - 100% = $0.00
- No payment information required for $0.00 checkout
- Subscription created with plan_type = 'standard', status = 'active'
- Coupon usage count increments in database
- User can access Standard plan features immediately
- No Square payment attempt made
{/traklet:section:expected}
