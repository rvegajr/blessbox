{traklet:section:objective}
## Objective
A super-admin can navigate to admin panel, click "Organizations" tab, view a list of all organizations with key statistics, and click on an organization to view details.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in as super-admin (via TC-022)
- Multiple organizations exist in database
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Sign in as super-admin (via TC-022)
2. Navigate to `/admin`
3. Verify admin panel loads with heading "Admin Panel"
4. Click "Organizations" button/tab
5. Verify organizations table displays with columns:
   - Organization name
   - Contact email
   - Registration count
   - QR code count
   - Subscription status
   - Created date
6. Click on an organization row
7. Verify organization details page loads showing:
   - Full organization information
   - Registration count and list
   - QR codes count
   - Subscription details
   - Membership count
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- All organizations from all accounts visible (not filtered by current user)
- Statistics are accurate and up-to-date
- Organizations sorted by created_at DESC by default
- Can sort by name, email, registration count, etc.
- Pagination if more than 50 organizations
- Details page shows comprehensive organization data
- Non-admin users cannot access `/admin` (redirected to login)
- Regular org admins cannot see other organizations
{/traklet:section:expected}
