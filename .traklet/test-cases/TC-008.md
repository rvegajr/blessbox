{traklet:section:objective}
## Objective
A user can download QR codes from the dashboard as PNG or PDF files, verify the files are high-quality and scannable, and print them for physical distribution.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in
- At least one QR code exists in organization
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to `/dashboard/qr-codes`
2. Find a QR code and click "Download" button
3. If format chooser appears, select PNG
4. Verify file downloads successfully
5. Open downloaded PNG file and verify:
   - Image is clear and high-resolution (512x512 or higher)
   - QR code is scannable with phone camera
6. Return to QR codes page
7. Download same QR code as PDF (if option exists)
8. Open PDF and verify:
   - QR code renders clearly
   - Entry point label/name included
   - Printable layout (fits on standard page)
9. Test scanning downloaded QR code with phone camera
10. Verify it opens correct registration URL
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- PNG files are high-resolution (minimum 512x512)
- PDF files include QR code + entry point label
- Downloaded files are named descriptively (e.g., "main-entrance-qr.png")
- QR codes remain scannable after download
- Multiple downloads of same QR code produce identical results
- No authentication required to access the scanned URL
- Download works in all major browsers
{/traklet:section:expected}
