{traklet:section:objective}
## Objective
An admin can reverse a check-in by clicking "Undo Check-in", verify the status reverts to "Not Checked In", and the registration can be checked in again.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in as org admin
- At least one registration with status "Checked In" (from TC-012)
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to registration detail page for checked-in registration
2. Verify status shows "Checked In" with timestamp
3. Locate "Undo Check-in" button
4. Click "Undo Check-in"
5. Verify status changes to "Not Checked In"
6. Verify timestamp disappears or shows "—"
7. Verify "Check In" button reappears
8. Test checking in again (verify it works)
9. Verify database updated:
   - checked_in_at set to NULL
   - checked_in_by cleared (if tracked)
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- Undo check-in clears checked_in_at timestamp
- Status reverts to "Not Checked In"
- "Check In" button becomes available again
- Can check in and undo multiple times
- Undo action logged in activity feed
- Dashboard statistics decrement check-in count
- No confirmation dialog required (or optional "Are you sure?" modal)
- Works from both list view and detail view
{/traklet:section:expected}
