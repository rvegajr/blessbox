{traklet:section:objective}
## Objective
During onboarding QR configuration step, an admin can define entry points (name, location, description), generate QR codes for each, preview them, and download as PNG or PDF.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in
- Form builder step completed (TC-003)
- User on QR configuration step
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Verify QR configuration page loaded
2. Fill entry point details:
   - QR Code Name: "Main Entrance"
   - Entry Point: "Main Door"
   - Location: "Front of Building"
   - Description: "Primary registration point"
3. Click "Add Entry Point" to add more:
   - "Side Door"
   - "Drive-Through Lane"
4. Click "Generate QR Codes"
5. Verify QR codes displayed for each entry point
6. Verify QR codes are scannable (clear, correct size)
7. Click "Download" for each QR code
8. Test download formats (PNG, PDF if available)
9. Scan downloaded QR code with phone camera
10. Click "Complete Setup" or "Finish"
11. Verify redirect to dashboard
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- QR codes generated immediately after form submission
- Each QR code contains unique URL: `/register/{orgSlug}/{qrSlug}`
- QR codes are clear, high-resolution, scannable
- Downloaded PNG files are 512x512 or higher
- Downloaded PDF contains QR code + entry point label
- QR codes stored in qr_codes table with organization_id link
- Scanning QR code opens registration form with correct entry point pre-selected
- Setup completion marks onboarding_completed = true
{/traklet:section:expected}
