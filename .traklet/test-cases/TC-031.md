{traklet:section:objective}
## Objective
An admin can add participants (potential class enrollees) to the organization by navigating to participants page, clicking "Add Participant", filling in details, and saving.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in as org admin
- Participants feature accessible
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to `/participants` or `/participants/new`
2. Click "Add Participant" button (if not already on creation page)
3. Verify participant creation form displays
4. Fill in participant details:
   - First Name: "Alice" (or "Alice {timestamp}")
   - Last Name: "Johnson"
   - Email: alice@example.com (or alice.{timestamp}@example.com)
   - Phone: (555) 111-2222
5. Click "Save Participant" button
6. Verify redirect to participants list `/participants`
7. Verify participants page heading: "Participants"
8. Verify new participant appears in list
9. Test adding second participant:
   - First Name: "Bob"
   - Last Name: "Smith"
   - Email: bob@example.com (unique)
10. Verify both participants in list
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- Form validates required fields (first name, last name, email)
- Email validation enforced (valid email format)
- Phone validation if required
- Participant created with unique ID
- Participant linked to current organization
- Participants list shows all org participants
- Cannot create duplicate participant (same email in same org) or shows warning
- Participants sorted alphabetically or by created_at
{/traklet:section:expected}
