{traklet:section:objective}
## Objective
A visitor can view pricing page at `/pricing`, see three subscription tiers (Free, Standard, Enterprise) with features and prices, and click "Get Started" or "Subscribe" to proceed to checkout.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- No authentication required (public page)
- Square sandbox configured if testing actual checkout
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to `/pricing`
2. Verify page loads (status 200)
3. Verify three plan tiers displayed:
   - Free Plan: $0/month
   - Standard Plan: $19/month
   - Enterprise Plan: $99/month
4. Verify each plan lists features (e.g., QR codes limit, registrations limit, storage, support)
5. Click "Get Started" or "Subscribe" button on Standard plan
6. Verify redirect to `/checkout?plan=standard`
7. Test clicking other plan buttons and verify correct plan query param
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- Pricing page is public (no authentication required)
- All three plans display clearly with prices
- Feature lists are complete and accurate
- Free plan includes a "Current Plan" or "Get Started" button
- Standard and Enterprise plans have "Subscribe" or "Upgrade" buttons
- Plan selection carries to checkout via query parameter
- Checkout page reflects selected plan and price
- Prices match database plan limits configuration
{/traklet:section:expected}
