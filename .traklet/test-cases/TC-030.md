{traklet:section:objective}
## Objective
An admin can add scheduled sessions to a class by navigating to the class sessions page, filling in date/time/duration/location, and saving the session.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in as org admin
- At least one class exists (from TC-029)
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to `/classes/{classId}` (class detail page)
2. Click "Add Session" button or navigate to `/classes/{classId}/sessions`
3. Verify session creation form displays
4. Fill in session details:
   - Date: 8 days from today (YYYY-MM-DD format)
   - Time: 10:00 AM
   - Duration: 60 minutes (or as available)
   - Location: "Main Studio"
   - Instructor: "Jane Smith" (optional)
5. Click "Save" button
6. Verify redirect back to class sessions page
7. Verify new session appears in sessions list
8. Verify session details are accurate
9. Test adding multiple sessions to same class
10. Verify sessions sorted by date/time
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- Form validates required fields (date, time)
- Date must be today or future date
- Time format accepted (10:00, 10:00 AM, etc.)
- Session created with unique ID
- Session linked to correct class_id
- Sessions list shows all sessions for class
- Can add multiple sessions on same day (different times)
- Sessions sorted chronologically
- Cannot create session in the past (or shows warning)
{/traklet:section:expected}
