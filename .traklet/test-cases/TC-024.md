{traklet:section:objective}
## Objective
A super-admin can navigate to `/admin/coupons`, view existing coupons, create new coupons with various discount types, edit coupons, and deactivate coupons.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in as super-admin (via TC-022)
- Coupon management page exists
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Sign in as super-admin
2. Navigate to `/admin/coupons`
3. Verify "Coupon Management" heading displayed
4. Verify existing coupons list (FREE100, WELCOME50, etc.)
5. Click "Create Coupon" or "Create New Coupon" link
6. Fill in coupon creation form:
   - Code: TEST2024 (unique alphanumeric)
   - Discount Type: Percentage (dropdown)
   - Discount Value: 25
   - Status: Active
   - Max Uses: 50
   - Expiration Date: 6 months from today
7. Click "Create Coupon" or "Save" button
8. Verify redirect back to coupon list
9. Verify new coupon appears in list
10. Click edit button on new coupon
11. Change max uses to 100
12. Save changes
13. Verify coupon updated
14. Test deactivating coupon (change status to inactive)
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- Coupon list displays all coupons system-wide
- Create form validates:
  - Code must be unique
  - Code must be alphanumeric (no spaces)
  - Discount value must be > 0
  - Percentage discount must be 1-100
  - Max uses must be positive integer or unlimited
- Created coupon immediately available for use at checkout
- Edit works for all coupon fields
- Deactivated coupons cannot be applied at checkout
- Cannot delete coupons with existing usage (or shows warning)
- Form has proper data-testids for automation
{/traklet:section:expected}
