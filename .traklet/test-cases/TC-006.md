{traklet:section:objective}
## Objective
A signed-in user can access `/dashboard` and view organization statistics (total registrations, QR codes, check-ins), recent activity, and navigation menu.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in with active organization membership
- Organization has completed onboarding
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to `/dashboard`
2. Verify page loads without errors (status 200)
3. Verify dashboard layout renders:
   - Organization name displayed
   - Statistics cards (registrations, QR codes, check-ins, etc.)
   - Recent activity feed
   - Navigation menu (sidebar or top nav)
4. Verify stat cards show accurate counts or "0" for new organizations
5. Click navigation menu items to verify routing works
6. Refresh page and verify dashboard persists (session maintained)
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- Dashboard requires authentication (redirects to `/login` if not signed in)
- Statistics display accurate real-time counts from database
- Recent activity shows last 10-20 events chronologically
- Navigation menu includes: Dashboard, Registrations, QR Codes, Classes, Participants, Settings, Admin (if super-admin)
- Loading states displayed while fetching data
- Error boundaries catch and display errors gracefully
{/traklet:section:expected}
