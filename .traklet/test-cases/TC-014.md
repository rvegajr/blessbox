{traklet:section:objective}
## Objective
An admin can select multiple registrations from the list using checkboxes, click "Check In Selected" or "Bulk Check-in", and check them all in at once.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in as org admin
- Multiple registrations exist with "Not Checked In" status
- UI includes checkbox selection feature
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to `/dashboard/registrations`
2. Verify checkboxes appear next to each registration row
3. Select 2-3 registrations by checking boxes
4. Verify selection count appears (e.g., "3 selected")
5. Locate "Check In Selected" or "Bulk Check-in" button
6. Click the bulk check-in button
7. Verify all selected registrations update to "Checked In"
8. Verify timestamps recorded for each
9. Verify checkboxes clear after operation
10. Verify can select and check in more registrations
11. Test "Select All" checkbox (if exists) to select all visible registrations
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- Checkbox selection works for individual rows
- "Select All" checkbox selects all visible (respecting pagination)
- Bulk check-in button only enabled when 1+ registrations selected
- All selected registrations update in single transaction
- If any check-in fails, either all roll back or partial success reported
- Loading state shown during bulk operation
- Success message shows count: "3 registrations checked in"
- Operation completes in under 5 seconds for 50 registrations
- Cannot bulk check-in already checked-in registrations (grayed out or filtered)
{/traklet:section:expected}
