{traklet:section:objective}
## Objective
An admin can export registrations as a PDF file, download it, open it, and verify it contains all registration data in a printable format.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in as org admin
- Organization has registrations
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to `/dashboard/registrations`
2. Locate "Export PDF" button (id="export-pdf")
3. Click export PDF button
4. Verify file download starts
5. Verify filename format: `registrations-{orgSlug}-{timestamp}.pdf`
6. Open downloaded PDF file
7. Verify PDF contains:
   - Organization name/logo (if configured)
   - Report title: "Registrations Report"
   - Generation date/time
   - Table with all registrations
   - Columns: Name, Email, Phone, Entry Point, Check-in Status, Date
8. Verify PDF is readable and well-formatted
9. Test printing PDF (preview print dialog)
10. Verify PDF header shows file signature: %PDF-1.x
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- PDF downloads successfully
- File opens in PDF readers (Preview, Adobe, Chrome)
- Data formatted in readable table layout
- Page breaks handle long lists gracefully
- Headers repeat on each page (if multi-page)
- Footer includes page numbers
- Print-friendly layout (fits on standard letter/A4 paper)
- File signature starts with %PDF (verified in spec)
- PDF metadata includes title, author, creation date
{/traklet:section:expected}
