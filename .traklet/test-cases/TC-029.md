{traklet:section:objective}
## Objective
An org admin can create a class/workshop by navigating to `/classes/new`, filling in class details (name, description, capacity, timezone), and saving it to the database.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in as org admin
- Classes feature enabled for organization
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to `/classes/new`
2. Verify class creation form displays
3. Fill in class details:
   - Name: "Yoga Basics" (or "Yoga Basics {timestamp}" for uniqueness)
   - Description: "Introduction to yoga for beginners"
   - Capacity: 20
   - Timezone: Select from dropdown (e.g., America/New_York)
4. Click "Create Class" button
5. Verify redirect to class detail page `/classes/{classId}`
6. Verify class appears in classes list at `/classes`
7. Verify class details are accurate
8. Check database: classes table has new record with correct data
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- Form validates required fields (name, capacity)
- Capacity must be positive integer
- Timezone dropdown includes common timezones
- Class created with unique UUID
- Class linked to current organization
- Redirect to class detail page after creation
- Class appears in org's classes list immediately
- Cannot create class with duplicate name in same org (or allows duplicate)
{/traklet:section:expected}
