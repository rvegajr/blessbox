{traklet:section:objective}
## Objective
A super-admin can navigate to admin panel, click "Subscriptions" tab, and view all subscriptions from all organizations with details including plan type, status, billing cycle, and amount.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in as super-admin (via TC-022)
- Multiple subscriptions exist across organizations
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Sign in as super-admin
2. Navigate to `/admin`
3. Click "Subscriptions" button/tab
4. Verify subscriptions table displays with columns:
   - Organization name
   - Plan type (Free, Standard, Enterprise)
   - Status (Active, Cancelled, Expired)
   - Billing cycle (Monthly, Yearly)
   - Amount (base price and discounted price if coupon used)
   - Next billing date
   - Created date
5. Verify subscriptions from all organizations visible
6. Test sorting by organization name, plan type, status
7. Filter by status (e.g., show only Active subscriptions)
8. Click on a subscription row (if detail view exists)
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- All subscriptions visible (not filtered by current organization)
- Accurate data for each subscription
- Can see which organizations are on which plans
- Status colors: Active (green), Cancelled (yellow), Expired (red)
- Amount shows both original and discounted price if coupon applied
- Billing cycle and next billing date accurate
- Pagination if more than 50 subscriptions
- Only super-admin can access this view
- Regular org admins cannot see other orgs' subscriptions
{/traklet:section:expected}
