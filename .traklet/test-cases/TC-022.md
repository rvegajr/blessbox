{traklet:section:objective}
## Objective
A user whose email matches SUPERADMIN_EMAIL can sign in by navigating to `/admin-login`, entering the configured SUPERADMIN_PASSWORD, and receiving a bb_session JWT with role: "superadmin" that grants access to `/admin` routes.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- SUPERADMIN_EMAIL environment variable set (e.g., admin@blessbox.app)
- SUPERADMIN_PASSWORD_HASH environment variable set (bcrypt hash)
- User account for SUPERADMIN_EMAIL exists or will be created
- No active session (logged out)
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to `/admin-login`
2. Verify page loads with login form (status 200)
3. Verify form contains:
   - Email input (`data-testid="input-admin-email"`)
   - Password input (`data-testid="input-admin-password"`)
   - Submit button (`data-testid="btn-submit-admin-login"`)
4. Enter SUPERADMIN_EMAIL in email field
5. Enter correct password in password field
6. Click submit button
7. Verify redirect to `/admin` within 3 seconds
8. Verify `/admin` page loads successfully (status 200)
9. Decode bb_session JWT and verify:
   - Payload contains `role: "superadmin"`
   - Payload contains `email: <SUPERADMIN_EMAIL>`
   - Token expiration set to 1 hour from now
10. Test protected admin routes are accessible (e.g., `/admin/coupons`)
{/traklet:section:steps}

{traklet:section:expected}
## Expected
**Success case:**
- Correct email + password → 200, bb_session cookie set, redirect to /admin
- JWT payload: `{sub, email, role: "superadmin", exp}`
- Cookie: httpOnly, sameSite: lax, secure (in prod), maxAge: 3600

**Error cases:**
- Wrong password → 401, no cookie, error message shown
- Wrong email (not super-admin) → 401, no cookie, error message
- Empty email or password → 400 validation error
- 5 wrong attempts from same IP in 60s → 429 rate limit error
- SUPERADMIN_PASSWORD_HASH not set → 404 (endpoint disabled)

**Security:**
- Rate limiting prevents brute force (5 attempts per IP per 60s)
- Password hashed with bcrypt (cost 12)
- Session expires in 1 hour (not 30 days like normal login)
- Endpoint returns 404 if SUPERADMIN_PASSWORD_HASH unset (never accidentally enables)
{/traklet:section:expected}
