{traklet:section:objective}
## Objective
An organization admin can view all registrations at `/dashboard/registrations`, see summary information in a table/list, and click on a registration to view full details.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in as org admin
- Organization has at least one registration (from TC-009)
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to `/dashboard/registrations`
2. Verify page loads with registrations list
3. Verify table/list displays columns:
   - Name (First + Last)
   - Email
   - Registration date/time
   - Entry point (QR code used)
   - Check-in status (checked in / not checked in)
   - View/Details button
4. Click "View" or registration row to see details
5. Verify detail page shows:
   - All form fields submitted
   - Registration timestamp
   - QR code/entry point used
   - Check-in status and timestamp (if checked in)
   - Check-in button (if not checked in)
6. Verify all submitted data is accurate
7. Return to list and verify can view other registrations
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- Only registrations for current organization displayed
- Most recent registrations appear first (sorted by created_at DESC)
- Pagination works if more than 50 registrations
- Loading state displayed while fetching data
- Empty state shown if no registrations yet
- Cannot view other organizations' registrations
- Check-in status updates in real-time
- Registration details page shows all custom field responses
{/traklet:section:expected}
