{traklet:section:objective}
## Objective
A new organization can complete initial setup by providing contact info, receiving a 6-digit verification code via email, entering it, and automatically signing in.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- Fresh browser session (no existing bb_session cookie)
- Valid SendGrid configuration (SENDGRID_API_KEY set)
- Email inbox accessible for verification code
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to `/onboarding/organization-setup`
2. Fill organization form:
   - Organization Name: "Test Food Bank"
   - Email: valid email address
   - Phone: (555) 123-4567
   - Address: 123 Test Street, Test City, TS 12345
3. Click Continue button
4. Verify redirect to email verification page
5. Check email inbox for 6-digit code
6. Enter the 6-digit code
7. Select organization type from dropdown
8. Click Continue
9. Verify redirect to form builder (next onboarding step)
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- Organization created in database with email_verified = 0 initially
- Verification code sent within 30 seconds
- Valid code → email_verified = 1, user signed in, bb_session cookie set
- Invalid code → error message, no sign-in
- Expired code (>10 minutes) → error message
- If user already signed in via magic link, code still works and continues onboarding
{/traklet:section:expected}
