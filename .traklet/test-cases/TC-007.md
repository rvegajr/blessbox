{traklet:section:objective}
## Objective
A user can navigate to `/dashboard/qr-codes`, view all QR codes created during onboarding or later, click on a QR code to see details, and verify scan count.
{/traklet:section:objective}

{traklet:section:prerequisites}
## Prerequisites
- User signed in
- Organization has at least one QR code (from TC-004)
{/traklet:section:prerequisites}

{traklet:section:steps}
## Steps
1. Navigate to `/dashboard/qr-codes` via navigation menu
2. Verify QR codes list displays
3. Verify each QR code shows:
   - Entry point name
   - Location
   - QR code thumbnail/preview
   - Scan count (should be 0 initially)
4. Click on a QR code
5. Verify details page loads with:
   - Full-size QR code image
   - Entry point details (name, location, description)
   - Scan count and scan history
   - Public registration URL
6. Copy registration URL and verify format: `/register/{orgSlug}/{qrSlug}`
{/traklet:section:steps}

{traklet:section:expected}
## Expected
- All organization QR codes displayed (filtered by organization_id)
- QR codes are clear and scannable
- Scan count increments when someone registers via that QR code
- Details page shows full information
- Public URL is shareable and works without authentication
- Cannot view other organizations' QR codes
{/traklet:section:expected}
