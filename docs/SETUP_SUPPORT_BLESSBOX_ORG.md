# 📧 Setting Up support@blessbox.org

## Step-by-Step Guide

### 1. Create New Sender in SendGrid

1. **Go to SendGrid Dashboard:** https://app.sendgrid.com
2. **Navigate to:** Settings → Sender Authentication → Single Sender Verification
3. **Click:** "Create New Sender" button (top right)
4. **Fill in the form:**
   - **From Email:** `support@blessbox.org`
   - **From Name:** `BlessBox Support`
   - **Reply To:** `rvegajr@yolovibecodebootcamp.com` (or your preferred email)
   - **Company Address:** Use the same address as your other senders
   - **Nickname:** `BlessBox Support`
5. **Click:** "Create"

### 2. Verify the Email

1. **Check your email inbox** for `support@blessbox.org` (if you have access)
2. **OR** if you control the domain, verify via DNS:
   - SendGrid will provide DNS records to add
   - Add them to your domain's DNS settings

### 3. Once Verified

The code is already updated to use `support@blessbox.org` - it will work automatically once verified!
